Comandos de GIT

PWD ==> Muestra donde estamos parados.
CD ==> Con este comando podemos movernos por las carpetas.
LS ==> Esto te muestra la lista de elementos.
MKDIR ==> Este comando te sirve para crear carpetas.
CAT ==> Sirve para leer y mostrar contenidos de archivos.
TOUCH ==> Este comando sirve para crear archivos.
NANO ==> Sirve para editar un archivo.
GIT INIT ==> Con este comando podes inicializar GIT en carpeta.
GIT ADD[archivo] ==> Agregar archivo a la version.
GIT ADD .[archivo] ==> Con el punto agregas todos los archivos a la version.
GIT COMMIT -M["mensaje"] ==> Con este comando podes agregar un comentario a tu archivo.
GIT PULL ==> Esto sirve para descargar version de repositorio remoto.
GIT PUSH ==> Esto es para subir verison a un servidor GIT.
GIT REMOTE ADD URL ==> Agregar repositorio remoto.
GIT -D "NOMBRE" ==> Sirve para crear una rama.
GIT CHECKOUT "NOMBRE" ==> Con este comando podes ir a la rama.
